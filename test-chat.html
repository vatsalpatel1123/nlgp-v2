Chat Widget Test Page

This is a test page to verify that the chat widget is working correctly.

Instructions:

  1. Look for the chat widget button in the bottom-right corner
  2. Click on the chat widget button to open the chat
  3. You should see a welcome message from Bella automatically
  4. Try typing a message and sending it

If you can see the chat widget and the welcome message appears when you open it, then everything is working correctly!

Open the browser console (F12) to see debug messages from the chat widget.
